Module: Icon

This is an icon component for both native and simple SVG icons

Props

Name Type Default value Required? Description
id Number - Yes Id of the icon component div
svgfile String - Yes Url of svg file
color Number 512 No color index value
size Number 55 No size value in `px`
x Number 0 No left-right margin
y Number 0 No top-bottom margin
isNative String 'false' No true for native svg icons

Data

Name Type Default value Description
idData Number - to change the id data
iconData String - to change the svgfile url data
sizeData Number 55 to change the size data
colorData Number 512 to change the color data
xData Number 0 to change the left-right margin data
yData Number 0 to change the top-bottom margin data
